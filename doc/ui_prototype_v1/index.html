E Echora V1 MVP 原型

Chat-First · Skills 驱动 · 流式交互

一个聊天输入框承载所有功能。AI 作为中央路由器，通过可插拔的 Skill 体系调度一切。 本目录是 V1 MVP 全部界面的 HTML 原型。

推荐浏览路径（新用户完整流程）
Step 1 登录 / 注册 极简登录页，邮箱注册 + JWT。登录后直接进入聊天界面。 auth Step 2 AI 对话式 Onboarding AI 通过对话收集用户姓名、年龄、年级、英语水平。完成后生成个性化画像。 onboarding Step 3 场景选择 · Scene Cards 基于用户画像推荐 4 个场景卡片 Widget。可点击选择或自定义输入。 scene-select Step 4 练习 · 填空模式 输入框切换为 fill 模式，句子模板 + 空位高亮。Tab 键切换空位。 practicefill Step 5 练习 · 翻译模式 整句汉译英，chat 模式输入框。展示 AI 出题卡片 + 用户提交答案。 practicetranslate Step 6 批改结果 · Grading Widget AI 流式批改结果卡片：得分、正确/错误标识、逐点解析、错误标签、快捷追问。 gradeexplain Step 7 复盘 · Progress Summary 对话式查看学习进度，AI 生成文字版总结 + 进步摘要 Widget。 reviewretry 交互细节 命令面板 · Command Mode 输入 / 触发命令浮层，模糊搜索，键盘导航。展示所有可用命令。 command
设计原则
Chat-First

整个应用就是一个聊天界面，所有功能通过对话承载。没有 Dashboard、没有多栏布局。

Skills 架构

所有功能封装为可插拔 Skill。AI 作为路由器，根据意图自动调用对应 Skill。

Widgets in Chat

复杂交互以 Widget 形式嵌入对话流：场景卡片、批改结果、进度摘要。

输入模式切换

输入框根据 Skill 自动切换：chat / fill / select / command 四种模式。
