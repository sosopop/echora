模式切换

由 SkillEvent · mode-switch 触发,前端从 onSkillEvent 中接收新模式名,切换 InputArea 子组件。

映射关系

  • onboarding / general-chat / explain / 翻译类题型 → chat
  • fill / 短语填空 → fill
  • scene-select / 选择填空 / intent-confirm → select
  • 用户点击 ☰ 按钮 → menu (临时叠加层,关闭后回到原模式)

键盘约定

  • chat: Enter 发送 · Shift+Enter 换行
  • fill: Tab / Shift+Tab 切空 · Enter (在最后一空) 提交
  • select: A/B/C/D 或 1/2/3/4 直选
  • menu: ↑↓ 选项 · Enter 触发 · Esc 关闭
