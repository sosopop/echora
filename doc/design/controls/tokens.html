切换规则
  • 默认跟随系统 @media (prefers-color-scheme: dark),无需用户操作
  • 用户点击顶栏 🌙 / ☀ 按钮可强制切换,写入 localStorage.echora-theme
  • 所有 token 以 CSS 变量定义,组件不硬编码色值 (PRD §4.2)
  • 语义色 (success / warning / error) 与品牌色 (primary) 在两个主题下保持一致
  • --color-hairline-dark: #2f2d2a 已补入 tokens.css 作为暗色分割线
