位置

悬浮于输入框左上方,距离输入区下边缘 8px。点击外部或 Esc 关闭。

状态适配

  • onboarding: 仅显示"了解你"流程相关项
  • scene_selecting: 推荐场景 / 自定义
  • practicing: 跳题 / 提示 / 难度调整
  • grading: 完全锁定,不显示菜单
  • awaiting_next / reviewing: 下一步引导 + 复盘动作

语义

菜单点击 = 结构化 action,不经过 AI Router 的 confidence 判断,直接走确定性路由。
