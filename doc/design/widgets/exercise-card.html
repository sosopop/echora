exercise-card

练习题主卡片。承载题干、上下文、提示与答题入口。根据题型嵌入 fill-blank 或 choice-question。

第 2 题 半句翻译 · 中等难度 politeness
场景:你坐在餐厅,服务员刚把菜单递过来。
把这一句礼貌地翻译成英文:
能给我看一下今日特餐吗?
💡 提示: 用 Could / Would 开头,语气更得体
在下方输入区作答 · 输入模式 · CHAT
第 3 题 单词填空 · 2 个空 collocation
把缺失的词补全:
I'd like a steak, _____, _____ .
💡 第 1 空: 熟度 (五分熟) · 第 2 空: 礼貌结尾词
输入模式 · FILL · Tab 切换空位
第 4 题 对话接龙 · 角色扮演
服务员对你说:
服务员: Would you like anything to drink with that?
你的回答 (用英文):
💡 任务: 点一杯饮料 + 询问是否有特价
输入模式 · CHAT · 自由作答

结构

  • widget-head: 题号 + 题型 + 核心知识标签
  • widget-body: 场景描述 + 题干 + 提示
  • widget-foot: 模式提示 + 跳过/更多提示按钮

题型映射输入模式

  • fill / 短语填空 → FILL 模式
  • 选择填空 → SELECT 模式
  • 翻译类 / 对话接龙 → CHAT 模式
