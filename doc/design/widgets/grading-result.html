结构

  • state-bar: 顶部 4px 颜色条 (绿/黄/红)
  • grade-head: 大字得分 + 状态徽章
  • compare: 你的答案 vs 参考表达
  • explain: 中文解析 (先肯定,再纠正,聚焦 1-2 点)
  • tags-row: 12 类错误标签 chip
  • grade-foot: 下一题 / 再说一次 / 追问入口

得分梯度

  • ≥ 90 ok 绿 + ✓ 答对
  • 60-89 warn 黄 + 轻微问题
  • < 60 bad 红 + 待加强
