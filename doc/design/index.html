对话即界面，
一位英语教练在等你练习。

本目录是 Echora 前端原型的导航入口。包含主要页面、12 种 Widget 组件、4 种输入区模式以及设计 Token 与基础控件库。所有原型遵循 DESIGN.md 的暖陶土色 + 纸张感视觉语言， 支持明 / 暗双主题 （右上角 🌙 / ☀ 切换或跟随系统），仅含必要的轻交互。

所有原型为静态 HTML + CSS,辅以最小 JS 实现 tab 切换、抽屉、popover 与主题切换。主题默认跟随 prefers-color-scheme, 手动选择写入 localStorage.echora-theme。 原型不承载真实业务逻辑,内容为占位文案,实现以 doc/prd.md 与 DESIGN.md 为准。
